Authentication: API anahtarınızı header ile gönderin: X-API-Key: YOUR_API_KEY

📦 Sipariş Oluşturma

POST /order/create
ParametreTürZorunluAçıklama
service_idintEvetServis ID'si
linkstringEvetSosyal medya linki
quantityintEvetMiktar
curl -X POST https://360medyasmm.com.tr/api/v1/order/create -H "X-API-Key: API_KEY" -d '{"service_id":10010,"link":"https://instagram.com/example","quantity":1000}'

🔍 Sipariş Sorgulama

GET /order/status/{order_id}
curl -X GET "https://360medyasmm.com.tr/api/v1/order/status/%23ABC123?api_key=API_KEY"
